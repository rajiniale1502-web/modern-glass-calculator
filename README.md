# Modern Glassmorphism Calculator

A sleek, highly responsive web calculator built from scratch using clean semantic HTML, modern CSS Grid, and vanilla JavaScript. Designed as part of an internship technical submission.

## ✨ Features
- **Glassmorphism UI:** Modern, translucent visual design utilizing CSS backdrop-filters and subtle ambient shadows.
- **Dual-Row Display:** Top row displays the active formula string while the bottom row dynamically renders calculations.
- **Real-Time Evaluation:** Calculates mathematical inputs on-the-fly as the user types or clicks.
- **Full Keyboard Support:** Fully mapped to standard computer keyboards (Numbers, Operators, Enter for `=`, Backspace for `DEL`, and Escape for `C`).
- **Input Guardrails:** JavaScript logic prevents consecutive broken operators (e.g., `++` or `*/`) to avoid application crashes.

## 🛠️ Tech Stack
- **HTML5:** Structuring layout layers and interactive components.
- **CSS3 (Grid & Flexbox):** Responsive button layout alignment and premium visual styling.
- **JavaScript (ES6):** State management, real-time string manipulation, and global keydown event listeners.
